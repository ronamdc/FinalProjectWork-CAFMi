***Queste cartelle servono a far funzionare il JSON della webapp CAFMi ***
Per poter sfruttare al 100% JSON fai questi passaggi:
- apri esplora risorse e vai su C:\
- estrai il contenuto di questo zip in C:\
- assicurati di avere la cartella JSON in C:\
Ora il gioco è fatto e hai integrato tutta le cartelle necessarie al linguaggio JSON per funzionare,
i rispettivi file si creeranno da soli nelle cartelle, puoi scegliere se tenerti le liste di base già qui dentro oppure cancellare tutti i file.


*** These folders are used to make the JSON of the CAFMi webapp ***
To take full advantage of JSON, follow these steps:
- open file explorer and go to C:\
- extract the contents of this zip to C:\
- make sure you have the JSON folder in C:\
Now the game is done! Once you have integrated all the folders necessary for the JSON language to work,
the respective files will create themselves in the folders, you can choose whether to keep the basic lists already here or to delete all the files.
